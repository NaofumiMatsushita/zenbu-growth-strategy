# 株式会社ワクリード様 × ZENBU株式会社
# 「あんしんサポート」提携企画書

**提案日**: 2026年1月
**提案者**: ZENBU株式会社 代表取締役 松下直史

---

## 提案の核心

ワクリード様の企業理念である**「縁ある人をワクワクにリードする」**、そして代表が掲げる**「暮らしに寄り添い続ける一番のパートナー」**という想いを、具体的なサービスとして実現する提携です。

---

## 1. なぜ今、この提携が必要なのか

### ワクリード様の現状課題（推察）

✓ オール浄水システムは競合他社も扱う商材
✓ 「売って終わり」では差別化が難しい
✓ 顧客との長期的な関係構築が成長の鍵
✓ リフォーム・不動産事業への顧客送客導線が必要

### あんしんサポートが解決すること

#### ① 圧倒的な差別化

```
【他社】
浄水器を販売して終わり

【ワクリード】
浄水器 + 10年間の住まいトラブル対応までセット
```

→ **「業界初!住まい全体を10年間サポートする浄水システム」**として訴求可能

#### ② "一番のパートナー"の具体化

- 水のトラブルだけでなく、電気・鍵・ガラス・建具まで対応
- 24時間365日、顧客に寄り添う体制
- 「助かった」「ありがとう」の声が増える接点創出

#### ③ LTV（顧客生涯価値）最大化

```
浄水器販売
   ↓
あんしんサポート加入
   ↓
トラブル対応時の接点創出
   ↓
エアコン・給湯器等のリフォーム提案
   ↓
不動産仲介へ
```

## 2. 商品設計とレベニューシェアモデル

### 商品ラインナップ

| プラン | 販売価格 | 想定ターゲット |
|--------|----------|----------------|
| **1年版** | 10,000円 | 既存顧客へのアップセル・お試し層 |
| **2年版** | 20,000円 | 新規顧客の標準プラン |
| **5年版** | 45,000円 | プレミアム志向層 |
| **10年版** | 90,000円 | 最上位プラン（浄水器とセット訴求） |

### 対象トラブル

| カテゴリ | 具体的なトラブル内容 |
|----------|---------------------|
| **水回り** | 蛇口の水漏れ、排水詰まり、トイレトラブル |
| **電気** | ブレーカー落ち、照明器具の故障、スイッチ不良 |
| **鍵** | 鍵紛失、鍵穴の不具合、鍵の開錠 |
| **ガラス** | 窓ガラスのひび割れ、破損 |
| **建具** | ドア・窓の建付け不良、網戸の破れ |

**対応時間**: 24時間365日受付
**対応エリア**: 三重県全域（将来的に拡大可能）
**費用**: 出張費・作業費無料（部品代のみ実費）

---

### レベニューシェア提案

#### パターンA: 40:60モデル

- **ワクリード様**: 40%（営業・販売・顧客管理）
- **ZENBU**: 60%（サービス提供・コスト負担・運営）

#### パターンB: 50:50モデル

- 初期段階のリスクシェア型
- データ蓄積後に比率見直し可能

---

### 月間収益シミュレーション（40:60モデル）

**前提条件**:
- 月間販売数: 50件
- 平均単価: 30,000円

**収益計算**:
```
月間売上: 1,500,000円

【ワクリード様分配】
600,000円/月 = 7,200,000円/年

【ZENBU分配】
900,000円/月 = 10,800,000円/年
```

---

### ワクリード様のメリット

✅ **初期投資ゼロ、在庫リスクゼロ**
✅ **既存の販売フローに組み込むだけ**
✅ **年間720万円の追加粗利**（しかも継続課金ではなくショット収益）
✅ **成約率向上による浄水器本体売上増**

## 3. 販売戦略提案

### Phase 1: 既存顧客へのテスト販売（1-3ヶ月）

**対象**: 既存顧客1,000名

**アプローチ方法**:
- 年1回の定期点検時に案内
- 「日頃の感謝を込めて特別価格でご案内」
- 1年版を特別価格7,000円で提供（通常10,000円）

**目標**:
- 200件の加入（転換率20%）
- 顧客反応のデータ収集
- トラブル発生率の初期データ取得

---

### Phase 2: 新規顧客への標準パッケージ化（4-6ヶ月）

| **プレミアム** | 浄水器 + あんしんサポート5年版 |
| **プラチナ** | 浄水器 + あんしんサポート10年版 |

---

#### 営業トークスクリプト例

```
「ワクリードでは、浄水器をお選びいただいたお客様に
10年間、住まい全体のトラブル対応をサポートしています。

水回りはもちろん、電気・鍵・ガラス・建具まで、

それが、ワクリードの『ワクワクを届ける』という想いです」
```

## 5. 成約率向上のインパクト試算

### 前提条件

- 現在の浄水器成約率: 30%（推定）
- あんしんサポート付帯後: 35%（+5%向上）
- 月間商談数: 150件
- 浄水器1台あたり粗利: 100,000円（推定）

---

### シミュレーション

#### 現状（サポートなし）

```
月間成約数: 150件 × 30% = 45件
月間粗利: 45件 × 100,000円 = 4,500,000円
```

#### サポート付帯後

```
月間成約数: 150件 × 35% = 52.5件（+7.5件）
月間粗利: 52.5件 × 100,000円 = 5,250,000円（+750,000円）

さらに、あんしんサポート売上（40%）: 600,000円

合計月間粗利増加: 1,350,000円
年間粗利増加: 16,200,000円
```

---

### 結論

**あんしんサポート提携により、年間約1,620万円の追加粗利が見込めます。**

## 7. リスク管理とZENBU側の対応

### 想定されるリスク

#### ① トラブル発生率が想定より高い

**対策**:
- 初期データで価格・プラン見直し
- 2年版中心の販売から段階的に長期プランへ

#### ② 顧客からのクレーム

**対策**:
- ZENBU側で24時間対応体制を構築
- ワクリード様への負担最小化

#### ③ 代理店展開時の品質管理

**対策**:
- 代理店向け研修プログラム提供
- 定期的な品質チェック体制
