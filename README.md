# ZENBU株式会社 長期成長戦略 2026-2031

## 概要

このリポジトリには、ZENBU株式会社の5ヶ年長期成長戦略計画が含まれています。

**計画期間**: 2026年 - 2031年
**重点領域**: DX（デジタルトランスフォーメーション）・テクノロジー活用
**目的**: 経営会議・社内共有

## ビジョン

**「不動産アフターサービスのデジタルプラットフォームNo.1企業へ」**

## 主要目標（2031年）

- 📈 **売上高**: 現在の5倍
- 🗺️ **対応エリア**: 全国主要都市10エリア以上
- 🏠 **管理戸数**: 100,000戸以上
- 💻 **デジタル化率**: 90%

## 4つのコア事業

1. **ソリューション事業** - 急ぎの問い合わせ対応、24時間メンテナンス、鍵交換代行
2. **コンサルティング事業** - 賃貸管理会社向け集客力向上、収益最大化支援
3. **カスタマーサクセス事業** - 建物維持管理、入居者対応代行、職人手配
4. **フィールド駆けつけサービス事業** - サブスク型修繕サービス

## 重点戦略

### 1. DX基盤の構築
全事業のデジタル化により、業務効率200%向上を実現

### 2. プラットフォーム化
顧客・職人・管理会社をつなぐデジタルプラットフォームの構築

### 3. データドリブン経営
予測型メンテナンスの実現とAI活用

### 4. エリア拡大
名古屋から全国主要都市への段階的展開

## DX戦略の3本柱

### フロントDX（顧客接点のデジタル化）
- 顧客ポータルアプリ
- チャットボット自動対応
- 管理会社向けダッシュボード

### バックエンドDX（業務プロセスのデジタル化）
- 業務管理システム
- 職人マッチング自動化
- データ基盤構築

### インフラDX（設備のIoT化）
- IoTセンサー導入
- 設備の遠隔監視
- 予防保全の実現

## 5年間のロードマップ

| 年度 | フェーズ | 重点施策 |
|------|----------|----------|
| 2026 | 基盤構築 | DX推進チーム設立、業務管理システム導入 |

| 2028 | エリア拡大 | 東海3県展開、AI予測メンテナンス |
