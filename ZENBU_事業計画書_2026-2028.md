## 1. エグゼクティブサマリー

### 1.1 事業の概要

ZENBU株式会社は、**AI技術を活用した騒音トラブル解決サービス**を提供します。賃貸住宅における騒音問題を科学的データで可視化し、管理会社・入居者・オーナーの三方良しを実現します。

### 1.2 市場機会

- **市場規模**: 日本の賃貸住宅市場は約30兆円
- **ターゲット**: 年間約18万件発生する騒音トラブル
- **成長性**: テレワーク普及で騒音トラブルが増加傾向（前年比125%）

### 1.3 競合優位性

| 項目 | 従来業者 | ZENBU |
|------|---------|-------|
| 対応時間 | 営業時間のみ | **24時間365日（AI）** |
| 初回対応 | 翌営業日 | **即時（AI自動）** |
| 価格 | 15,000円〜 | **8,800円〜（40%安）** |
| スケーラビリティ | 人員比例 | **AI自動（無制限）** |
| データ蓄積 | 紙・個別管理 | **クラウド一元管理** |

### 1.4 財務ハイライト

**3年間の成長計画（保守シナリオ）:**

| 年度 | 売上高 | 営業利益 | 営業利益率 |
|------|--------|----------|-----------|
| 2026年度 | 3,600万円 | △800万円 | -22% |
| 2027年度 | 9,600万円 | 2,400万円 | 25% |
| 2028年度 | 2.4億円 | 8,000万円 | 33% |

**累積**: 売上3.36億円、営業利益9,600万円

### 1.5 資金調達

- **必要額**: 3,000万円
- **使途**: システム開発（1,500万円）、マーケティング（1,000万円）、運転資金（500万円）
- **調達方法**: VC投資 or エンジェル投資 or 銀行融資

### 1.6 Exit戦略

- **2028年**: 月間売上3,000万円達成時にM&A検討
- **候補先**: 不動産管理大手（大東建託、東建コーポレーション等）
- **想定評価額**: 売上倍率3-5倍 = 7.2億円〜12億円

## 3. 市場分析

### 3.1 市場規模

#### TAM（Total Addressable Market）: 全体市場
```
日本の賃貸住宅戸数: 1,906万戸（2023年）
× 年間トラブル発生率: 3%
× 平均解決コスト: 3万円
= 171億円
```

#### SAM（Serviceable Available Market）: 獲得可能市場
```
都市圏（東京・大阪・名古屋）賃貸戸数: 800万戸
× 年間トラブル発生率: 3%
× 測定サービス利用率: 30%
× 平均単価: 1.2万円
= 86億円
```

#### SOM（Serviceable Obtainable Market）: 実獲得市場
```
Year 1: 0.1%シェア = 860万円
Year 2: 0.3%シェア = 2,580万円
Year 3: 1.0%シェア = 8,600万円
```

### 3.2 市場トレンド

#### 成長ドライバー

1. **テレワーク常態化**
   - 在宅時間増加 → 騒音への感度上昇
   - 2020年比で騒音クレーム125%増（国土交通省調査）

2. **高齢化社会**
   - 音への敏感さ増加
   - 単身高齢者の賃貸入居増

3. **法整備の進展**
   - 受忍限度の基準明確化
   - 客観的データの重要性向上

4. **デジタル化・AI普及**
   - オンライン完結サービスへの抵抗感減少
   - AI活用サービスの一般化

#### 市場の課題

1. **認知度の低さ**
   - 騒音測定サービスの存在を知らない: 82%
   - 泣き寝入りする入居者: 68%

2. **価格の不透明性**
   - 事前見積もりなし: 60%
   - 高額請求の不安: 75%

3. **効果への懐疑**
   - 「測定しても解決しない」: 55%
   - 「お金の無駄」: 43%

**→ ZENBUの機会**: 透明性・迅速性・効果の可視化で解決

### 3.3 顧客セグメント

#### プライマリターゲット（B2C）

**セグメント1: 20-30代単身者（都市部）**
- 人口: 約150万人（東京23区）
- 特徴: デジタルネイティブ、SNS活用、権利意識高い
- ペイン: 上階の足音、深夜騒音
- 年間想定利用者: 4.5万人

**セグメント2: 30-40代ファミリー（郊外）**
- 人口: 約200万世帯
- 特徴: 子育て世代、騒音被害・加害両面
- ペイン: 子供の声、楽器音
- 年間想定利用者: 3万世帯

**セグメント3: 60代以上シニア**
- 人口: 約180万人
- 特徴: 音に敏感、訴訟も辞さない
- ペイン: 生活音全般
- 年間想定利用者: 2.7万人

#### セカンダリターゲット（B2B）

**セグメント4: 中小不動産管理会社**
- 社数: 約8,000社
- 管理戸数: 平均500戸
- ペイン: クレーム対応の人員不足
- 想定契約社数: 200社（Year 3）

**セグメント5: 大手管理会社**
- 社数: 約50社
- 管理戸数: 平均5万戸
- ペイン: 訴訟リスク、ブランド毀損
- 想定契約社数: 5社（Year 3）

### 3.4 競合分析

#### 直接競合

**A社（全国展開の騒音測定業者）**
- 強み: 実績豊富、全国対応
- 弱み: 価格高い（2万円〜）、対応遅い、アナログ
- シェア: 推定15%

**B社（地域密着型）**
- 強み: 地元での信頼
- 弱み: スケール不可、属人的
- シェア: 推定5%（地域限定）

**C社（新興スタートアップ）**
- 強み: デジタル化
- 弱み: 実績不足、技術力
- シェア: 推定2%

#### 間接競合

- **弁護士**: 高額（30万円〜）、時間かかる
- **探偵業者**: 証拠収集のみ、分析なし
- **DIY測定**: 精度低い、法的効力なし

#### ZENBU の差別化ポイント

1. **AI完全自動化** → コスト競争力
2. **24/365即時対応** → 顧客満足度
3. **透明な価格** → 信頼性
4. **データ蓄積** → 継続的価値提供
5. **音源AI推定** → 技術優位性

### 3.5 市場参入障壁

#### 参入障壁（競合が真似しにくい理由）

1. **AI技術**: 音源推定モデルの学習データ蓄積
2. **スケールメリット**: 案件数増でも限界コスト低い
3. **ブランド**: 先行者利益でデファクト化
4. **ネットワーク効果**: データ増→精度向上→顧客増
5. **提携関係**: 管理会社・弁護士との独占契約

#### 退出障壁（撤退リスク）

- 初期投資3,000万円のうち、システム開発費1,500万円は固定費
- ただし、SaaS転換やM&Aで回収可能
