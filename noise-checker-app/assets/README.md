# アセットファイル

このディレクトリには以下のアセットファイルを配置してください：

## 必須ファイル

### 1. icon.png
- サイズ: 1024x1024 px
- 用途: アプリアイコン
- フォーマット: PNG (透過背景推奨)

### 2. splash.png
- サイズ: 1242x2436 px
- 用途: スプラッシュスクリーン
- フォーマット: PNG
- 背景色: #0a0e27 (ダークネイビー)

### 3. adaptive-icon.png
- サイズ: 1024x1024 px
- 用途: Android用アダプティブアイコン
- フォーマット: PNG
- 注意: 中央の512x512pxが表示されるため、重要な要素は中央に配置

### 4. favicon.png
- サイズ: 48x48 px
- 用途: Webファビコン
- フォーマット: PNG

## デザインガイドライン

### カラーパレット
- プライマリ: #4fc3f7 (明るいブルー)
- 背景: #0a0e27 (ダークネイビー)
- アクセント: #ef5350 (レッド)

### アイコンデザイン
- シンプルで認識しやすいデザイン
- 音波や騒音メーターをモチーフに
- ダークモードに対応した配色

## プレースホルダー画像の生成

Figma、Sketch、Adobe XDなどのデザインツールを使用してください。

または、以下のオンラインツールを利用できます：
- https://www.canva.com/ (無料)
- https://www.figma.com/ (無料)
- https://www.pixlr.com/ (無料)

## サンプルコンセプト

騒音チェッカーアプリなので、以下の要素を含めると良いでしょう：
- 音波のグラフィック
- マイクのアイコン
- dBメーター
- シンプルでモダンなデザイン
